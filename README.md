# Kaffe Stuga — Home-Style Swedish Café (Harris, MN)

A single-page website concept for **Kaffe Stuga**, a family-run, home-style café in
Harris, Minnesota that has served the area for over 50 years.

🔗 **Live site:** https://elitanz.github.io/kaffe-stuga/

## About

Kaffe Stuga ("coffee cottage") is a real local business currently without a website —
it operates through a Facebook page and review listings. This is an independent concept
site built from publicly available information to show what a modern web presence could
look like.

- **Address:** 43821 Forest Blvd, Harris, MN 55032
- **Phone:** (651) 674-9958
- **Hours:** Mon–Thu 7:30am–7:00pm · Fri–Sat 7:30am–7:30pm · Sun 8:00am–2:00pm

## Features

- Single static `index.html` — no build step, instant load
- Live, cursor-reactive **Three.js** hero (roasted coffee beans that stir around your
  cursor, with a warm light that follows the mouse) using image-based lighting + ACES
  tone mapping for a photorealistic look
- Pinned scroll-telling section, parallax gallery, count-up stats, and cursor-spotlight
  feature cards
- Fully responsive with a fast photo fallback on mobile; all motion respects
  `prefers-reduced-motion`
- SEO-ready: meta description, Open Graph tags, and `Restaurant` JSON-LD structured data
- Every button works (`tel:` calling, Google Maps directions, Facebook link)

## Notes

Food photography is illustrative (via [Unsplash](https://unsplash.com)). Business name,
address, phone, hours, and review quotes are drawn from public listings
(Tripadvisor / Restaurantji) and should be confirmed by the owner before going live.
